Fact 1.0.1.

Problem 1. Sei \(f\colon \mathbb{R}^n \rightarrow \mathbb{R}\) eine Funktion. Gesucht ist eine Minimalstelle von \(f\) unter drei Arten von Nebenbedingungen:
🔗
  1. Ungleichungsnebenbedingungen: \(g_i(x) \leq 0 \, \forall i \in \{1, \ldots, p\}\text{,}\) für \(g_i\) reelwertige Funktionen auf \(\mathbb{R}^n\text{.}\)
    🔗
    🔗
  2. Gleichungsnebenbedingungen: \(h_j(x) = 0 \, \forall j \in \{1, \ldots, m\}\text{,}\) für \(h_j\) reelwertige Funktionen auf \(\mathbb{R}^n\text{.}\)
    🔗
    🔗
  3. Abstrakte Nebenbedingungen: \(x \in C\) für \(C \subseteq \mathbb{R}^n\) abgeschlossen.
    🔗
    🔗
